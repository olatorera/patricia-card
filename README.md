# patricia-card
This project was created with Vue.js
The app displays the details on an ATM card
## Project Setup
- clone the repo  [https://github.com/olatorera/patricia-card.git](https://github.com/olatorera/patricia-card.git)
- ```bash
  npm install
  ```
- ```bash
  npm run serve
  ```
